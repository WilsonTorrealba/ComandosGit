Comandos de git da +Devs2Blu

cd : entrar na main
git init :
git clone : clonar repositório remoto no local
git add . : adicionar alterações
git status : ver estado do repositorio local
git commit -m "decrição" : para commitar
git push : empurrar as alterações commitadas para o repositório remoto
git checkout : Volta para o último commit
git checkout -b : para criar uma nova branch
git push origin : empurrar na branch escolhida
git checkout branch :
git merge : para mesclar branches, deve estar na main para executar esse git
git pull : atualizar repositório local
git tag : listar as tags criadas
git log : para ver os ultimos commits
git tag : criar tag nova
git tag -m "descriçãoDaTag" : criar tag nova com descrição
git tag -n : listar as tags + descrição
git push origin : empurrar tag para o repositório
git push origin --tag : empurrar todas as tags pro repositório
git show : obtém informações sobre a Tag
git tag -d : deletar tag
git push -delete origin : deletar tag no repositório remoto
